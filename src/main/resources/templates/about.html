This application is quick a demo for Mongo DB seach query translator.

Feel free to modify and test this application in any way you want, but remember it is not production ready ;)

Application comes with built in Mongo Database. If you want to connect to this database with external tools, connection string was outputed in application log.

If you want to learn more about translator, or add it to your own project, please follow instructions in transtalor repository.

If you have any suggestions or issues, (either to this application or translator itself) please create ticket in matching GitHub project.
